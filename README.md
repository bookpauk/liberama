## Сборка проекта
Необходима версия node.js не ниже 16.

```
$ git clone https://github.com/bookpauk/liberama
$ cd liberama
$ npm i
```

### Windows
```
$ npm run build:win
```

### Linux
```
$ npm run build:linux
```

Результат сборки будет доступен в каталоге `dist/linux|win` в виде исполнимого (standalone) файла

### Разработка
```
$ npm run dev
```
